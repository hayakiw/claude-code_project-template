# Claude Code プロジェクト設定

## 推奨する構成
```
your-project/
├── CLAUDE.md                    # Claude用プロジェクト情報（セッション開始時に自動読み込み）
├── .mcp.json                    # MCPサーバー設定（外部ツール連携）
├── .claude/
│   ├── .gitignore               # 個人設定ファイルを除外
│   ├── agents/                  # サブエージェント（特定タスク専門）
│   ├── skills/                  # オンデマンドで読み込まれる（タスクに関連する場合のみ）
│   ├── commands/                # スラッシュコマンド（/onboard など）
│   └── rules/                   # 常に適用すべきルール（コーディング規約など）
└── docs/                        # 要件定義書、設計書等
```

## 各ファイル・ディレクトリの説明

### CLAUDE.md
- プロジェクトの概要、アーキテクチャ、重要なコマンドを記載
- 全セッションで自動的に読み込まれる

### .claude/agents/
- 特定タスク専門のサブエージェント
- セッション開始時に自動読み込み
- 例：`code-reviewer.md` - コードレビューエージェント、`test-writer.md` - テストコード作成エージェント

### .claude/rules/
- 常に適用されるルール・制約
- 例：コーディング規約、レビュー基準

### .claude/skills/
- プロジェクト固有のスキル定義
- 関連するタスク時のみ自動で読み込まれる
- 例：`coding-standards/SKILL.md`

### .claude/commands/
- カスタムスラッシュコマンド
- 例：`/onboard` - 新メンバーのオンボーディング

### .mcp.json
- 外部ツール連携設定（GitHub、JIRA等）
- オプション（使用する場合のみ）
